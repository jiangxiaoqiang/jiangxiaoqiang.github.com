部署Key

首先必須用ssh-keygen製作一個SSH Key，供GitHub當作Deploy key使用。

當 SSH key 製作完成後，複製 Public key 到 GitHub 上的 Deploy key 欄位。這裡由於目錄下本身已經有原來的GitHub的rsa的key，所以生成的文件名為id_rsa.travis。
